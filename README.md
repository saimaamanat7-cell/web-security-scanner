# Web Security Scanner

A lightweight Python tool to quickly check websites for common security vulnerabilities.

## Features
- 🔐 SSL & website availability check
- 🛡️ Security header analysis
- 🧪 Reflected XSS detection
- 💉 SQL Injection testing (including time-based attacks)

## How to Run
1. Clone the repository:

git clone https://github.com/saimaamanat7-cell/web-security-scanner.git

2. Go to the folder:

cd web-security-scanner

3. Run the scanner:

python scanner.py

4. Check `result.txt` for output.

## Notes
- Current version focuses on server-side checks.
- JavaScript-based XSS requires browser-level analysis (future update).
